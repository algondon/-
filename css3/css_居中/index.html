水平居中

父元素相对定位，子元素绝对定位

通用居中方式

        .parent1{
          position:relative;
          background-color: yellow;
        }
        .children1{
          background-color: green;
          position:absolute;
          left:50%;
          transform: translateX(-50%);
        }
子元素采用absolute会导致没有将parent撑起，单独新建一个div将其撑起

flex布局

        .parent2{
          display:flex;
          justify-content: center;
          background-color: yellow;
        }
        .children2{
          background-color: green;
        }

居中的元素为常规文档流中的内联元素(display: inline)

.parent2{ text-align:center; }常见的内联元素有：span, a, img, input, label 等等

居中的元素为常规文档流中的块元素(display: block)

方式一：设置 margin

常见的块元素：div, h1~h6, table, p, ul, li 等等 
           .parent4{
             background-color: #33ff33;
           }
           .children4{
             margin:0 auto;
             width:300px;
             background-color: skyblue;
           }

方式二：修改为 inline-block 属性

常见的块元素：div, h1~h6, table, p, ul, li 等等

           .parent5{
             background-color: #33ff33;
             text-align: center;
           }
           .children5{
             display:inline-block;
             background-color: skyblue;
           }

居中的元素为浮动元素

           .parent6{
             background-color: #33ff33;
           }
           .children6{
             float:left;
             background-color: skyblue;
             position:relative;
             left:50%;
             transform:translateX(-50%);
           }

居中的元素为绝对定位元素

           .parent7{
             background-color: #33ff33;
             position:relative;
           }
           .children7{
             background-color: skyblue;
             position:absolute;
             left:50%;
             transform:translateX(-50%);
           }

垂直居中

父元素相对定位，子元素绝对定位

通用垂直居中方式

        .parent21{
          position:relative;
          background-color: #33ff33;
        }
        .children21{
          background-color: skyblue;
          position:absolute;
          top:50%;
          transform: translateY(-50%);
        }
子元素采用absolute会导致没有将parent撑起，单独新建一个div将其撑起

flex布局

适用于子元素为浮动，绝对定位，内联元素，均可垂直居中。

        .parent22{
          display:flex;
          align-items:center;
          background-color: #33ff33;
        }
        .children22{
          background-color: skyblue;
        }

利用flex-item的margin自动填充对于区域实现

      .parent22{
        display:flex;
        align-items:center;
        background-color: #33ff33;
      }
      .children22{
        background-color: skyblue;
